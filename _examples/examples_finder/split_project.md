# Split Project

Меню Sublime Text:

`Tools → Qsp tools → Splitters → Split Project`

**Как пользоваться:**

1. Откройте в ST любой файл "`.qsp`" из этой папки.
2. В меню `Tool` Сублайм Текста выберите последовательно следующие пункты:
	- `Qsp tools` 
	- `Splitters`
	- `Split Project`
3. В папке появятся подпапки одноимённые файлам с расширением "`.qsp`", в каждой из которых будут текстовые файлы qsps по одному на каждую локацию в игре.

Точно так же вы можете сплитить любые свои проекты, которые до сих пор разрабатывали в Quest Generator.
