# QSP to qsps

Эти файлы предназначены для проверки работы конвертера. Не самая сложная игра, однако она как нельзя хорошо показывает, соответствует ли формат текущего встроенного конвертера формату конвертера TXT2GAM.

`ukuzya_old.qsp` - оригинальная игра без изменений
`ukuzya_txt2gam.qsps` - так выглядит игра при конвертировании из QGen утилитой TXT2GAM

При конвертировании встроенной утилитой плагина все отличия сводятся к первым четырём строкам в файле qsps. То есть если из результирующего файла убрать первые четыре строки, в остальном файлы должны полностью совпадать, либо отличаться только промежутками между локациями.

Меню Sublime Text:

`Tools → Qsp tools → Converters (single files) → Game to QSPS`

**Как пользоваться:**

1. Откройте файл `ukuzya_old.qsp` в ST.
2. В меню `Tool` Сублайм Текста выберите последовательно следующие пункты:
	- `Qsp tools`
	- `Converters (single files)` 
	- `Game to QSPS`
3. В папке появится файл `ukuzya_old.qsps`, сконвертированный в формат TXT2GAM.

Таким образом вы можете сконвертировать любую игру в формат TXT2GAM.
