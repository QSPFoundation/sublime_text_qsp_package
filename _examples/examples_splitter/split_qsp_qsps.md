# Split Current File

Меню Sublime Text:

`Tools → Qsp tools → Splitters → Split Current File`

**Как пользоваться:**

1. Откройте в ST любой файл "`.qsp`" или "`.qsps`" из этой папки.
2. В меню `Tool` Сублайм Текста выберите последовательно следующие пункты:
	- `Qsp tools` 
	- `Splitters`
	- `Split Current File`
3. В папке появится подпапка одноимённая открытому вами файлу, а в этой папке будут текстовые файлы qsps по одному на каждую локацию в игре или файле "qsps", в зависимости от того, какой файл вы открывали.

Точно так же вы можете сплитить любые файлы игр или общие файлы формата TXT2GAM, сконверченные сторонними утилитами.
